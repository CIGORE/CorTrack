Metodologi & Tentang Platform

Cara kerja platform, batasan data, dan prinsip transparansi

Metodologi Pengumpulan Data

Platform ini mengacu pada data resmi yang tersedia untuk publik dari lembaga pemerintah Indonesia. Berikut prinsip pengumpulan data:

  • Hanya data yang secara sah diumumkan untuk publik yang ditampilkan
  • Data tidak tersedia publik tidak diisi, dikarang, atau diasumsikan
  • Setiap data menyertakan atribusi sumber dan tanggal update
  • Red flag/indikasi risiko bersifat statistik, bukan kesimpulan hukum
  • Data historis menggunakan arsip resmi yang terverifikasi
  • Coverage data per tahun ditampilkan secara transparan
Batasan Platform
  • Platform tidak menyediakan data real-time absolut; data bersifat near real-time atau update periodik sesuai jadwal sumber
  • Dataset KPK untuk kasus korupsi tersedia mulai 2003; data 2000–2002 terbatas
  • Data LHKPN hanya untuk penyelenggara negara yang wajib lapor
  • Data pengadaan mencakup paket yang terdaftar di SiRUP/SPSE; pengadaan di luar sistem tidak termonitor
  • Data pendidikan pejabat sangat terbatas karena tidak semua profil resmi memuat informasi ini
  • Skor risiko adalah alat bantu analitik, bukan kesimpulan investigasi
Tentang INTEGRITAS

Platform INTEGRITAS adalah prototipe sistem monitoring antikorupsi yang bertujuan meningkatkan transparansi tata kelola pemerintahan Indonesia melalui visualisasi data resmi yang mudah dipahami publik. Platform ini dirancang untuk menjadi alat bantu jurnalis, peneliti, akademisi, lembaga antikorupsi, dan masyarakat sipil dalam memantau tata kelola pemerintahan secara berbasis data.

Platform ini merupakan prototipe frontend yang dirancang untuk dikembangkan lebih lanjut dengan integrasi API resmi, backend data pipeline, dan sistem autentikasi untuk fitur watchlist dan alert.
