Cara kerja platform, sumber data, dan panduan penggunaan

Metodologi Pengumpulan Data

Platform mengacu data resmi publik dari lembaga pemerintah Indonesia:

  • Hanya data yang sah diumumkan publik yang ditampilkan
  • Data tidak tersedia publik tidak diisi atau diasumsikan
  • Setiap data menyertakan atribusi sumber dan tanggal
  • Red flag bersifat statistik, bukan kesimpulan hukum
  • Coverage data per tahun ditampilkan transparan

📖 Panduan Penggunaan Platform

Selamat datang di INTEGRITAS. Panduan singkat setiap modul:

🏠 Dashboard
  • Ringkasan angka utama di stat cards atas
  • Tren kasus per tahun 2003–2026
  • Tabel Kasus Landmark — klik "Lihat Semua"
  • Top 5 provinsi risiko tertinggi
📋 Monitoring Kasus
  • Filter: Status · Tahun (2000–2026) · Instansi
  • Klik Detail (🔍) untuk kronologi lengkap
  • Klik Reset untuk hapus semua filter
  • Pagination 10 kasus per halaman
👤 Pejabat & LHKPN
  • Data dari e-LHKPN KPK publik
  • Filter "Hanya Berindikasi" untuk red flag
  • ⚠ kuning = pemantauan; 🔴 = anomali LHKPN
  • Klik Detail untuk rincian komposisi aset
🗺️ Peta Wilayah (38 Provinsi)
  • Klik baris provinsi → tampil kabupaten/kota
  • Filter: Pulau/Wilayah · Level Risiko
  • Cari nama provinsi, ibu kota, atau kabupaten
  • Data 38 provinsi termasuk 4 DOB Papua
📦 Pengadaan
  • Filter "Hanya Red Flag" untuk anomali
  • Badge merah = pola statistik mencurigakan
  • Klik Detail untuk daftar indikasi per paket
  • Sumber: SiRUP, SPSE, Inaproc (LKPP)
⚡ Risk Center
  • Skor 0–100: Kritis >80 · Tinggi 60–80
  • ⚠ Skor = sinyal analitik, BUKAN tuduhan
  • Entitas skor tinggi perlu verifikasi primer
  • Update mengikuti perkembangan kasus
🔍 Global Search (tekan "/")
  • Tekan / di keyboard → langsung ke search
  • Pencarian serentak semua kategori data
  • Minimal 2 karakter untuk mulai cari
  • Hasil dikelompokkan per kategori
🌙 Mode Gelap/Terang
  • Klik ikon bulan/matahari pojok kanan atas
  • Di HP: ikon di pojok kanan mobile topbar
  • Preferensi tersimpan otomatis di browser
  • Menu hamburger (☰) untuk akses sidebar
⚠ Penggunaan Bertanggung Jawab
Platform ini hanya menampilkan data resmi pemerintah. Seluruh data tersangka yang belum inkracht tunduk pada asas praduga tak bersalah. Semua temuan wajib diverifikasi via sumber primer sebelum dipublikasikan. Sumber resmi: kpk.go.id · kejaksaan.go.id · elhkpn.kpk.go.id · lkpp.go.id
